Interface IHPStateManager

Namespace: RandomizerMod.RC.StateVariables
Assembly: RandomizerMod.dll
Syntax
public interface IHPStateManager
Remarks

This interface only uses return values, rather than the ref pattern. This is because many hp operations may require branching to fully account for how their effects interact with the hp state model.

Automatically fails if used when any focus charm (Joni, Deep Focus) is not determined.
Automatically fails if used with indeterminate hp. See IsHPDetermined(LazyStateBuilder).
